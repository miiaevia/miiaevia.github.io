Java Method Declaration

When declaring a method in Java, it is important to tell the program whether or not the method will be returning any values. Values can be returned in the form of any acceptable value type, such as a string, an integer, a double, etc. However, not all methods will return a value type, because it is not always necessary.

If the program will need to take in value from another method, that method will need to be declared with the value type. For example, if a method will be returning a string value, such as a name, the method would be written as follows: public static String MethodName(). Please note that any parameters the method needs to take in would be listed in the parentheses.

If no return type is needed, then the method is declared as void, meaning that there is no return type. In this example, the method would be declared as follows: public static void MethodName().
