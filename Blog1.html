Java Scanner Tool

When wanting to accept user input in a Java program, it is common for developers to import the java.util.Scanner tool. But what does it mean to "import" this tool?

The java package is a universal file available to Java programers that contains several packages. A "package" is a container for a class, which means it holds the predefined tools we wish to import.

The util folder lives inside the java folder and contains utility "classes", such as the Scanner, a class that was created to accept user input.
